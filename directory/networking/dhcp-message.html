DHCP Messages

Introduction

DHCP is service that comes with its networking protocol In this instance we are going to focus on the networking side of the service. It has the PORT 53 UDP and TCP. And its purpose is to assign IP addresses automatically.

Please select the specific message you want to learn more about

Normal DHCP MESSAGE
DHCPNAK DHCPDECLINE
DHCPRELEASE DHCPINFORM

Pros.

  • Required less individual configuarion
  • Reduces OpEx (Operation Expenses)

Cons.

  • Can be tempered by attacks such as DCHP Spoofing
